# ExpotecFrontend  

## Desenvolvimento  
#### Node version == v6.10.0.  

Instale o Gulp e o Bower globalmente.  
```npm install -g gulp bower```  
Instale os pacotes do node.  
```npm install```  
Instale os pacotes do Bower.  
```bower install```  

## Como rodar o projeto  

Iniciar o server:  
```npm start```  

Buildar o projeto:    
```npm build```  

Limpar o projeto:  
```npm run clean```
