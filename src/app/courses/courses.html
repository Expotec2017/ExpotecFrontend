Qual o objetivo do CITIC?

Contribuir para a discussão e compartilhamento de inovações tecnológicas, desenvolvimento pessoal, integração com o mercado de trabalho e interdisciplinaridade das temáticas envolvendo a computação.

Desenvolvimento Webapps com banco noSQL 11/08

  • Palestrante: Willian Carvalho
  • Horário da Palestra: 19:30
  • Descrição: Neste minicurso irei demonstrar como utilizar um banco de dados noSQL para realizar registros de dados através de uma app da plataforma Android utilizando os recursos disponíveis no Firebase.
